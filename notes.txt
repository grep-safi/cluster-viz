In D3, to create multiple svg elements, you must use a join with data.

For example, the following creates a single circle with a specified x, y, and radius:

svg.append("circle")
    .attr("cx", d.x)
    .attr("cy", d.y)
    .attr("r", 2.5);


But if you wanted multiple circles, you could create a data object, and use the 'selectAll'
function to bind the data to circle objects. But circle objects don't exist yet. They are
appended as data is used to enter new elements

svg.selectAll("circle")
  .data(data)
  .enter().append("circle")
    .attr("cx", function(d) { return d.x; })
    .attr("cy", function(d) { return d.y; })
    .attr("r", 2.5);
